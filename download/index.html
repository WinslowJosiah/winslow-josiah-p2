How To Use

The Python interpreter takes arguments from the command-line, and it is run like this:

gematrimax.py [ -i input-file ] [ -c ] gematrimax-program

gematrimax-program is the program file which you want to run as Gematrimax code. (It can have any extension, though .gmx is recommended.)

The options are optional, but if included, they do the following:

  • -i / --input: Take input from a specified file, instead of STDIN.
  • -c / --cheatmode: Interpret program in Cheat Mode instead of Gematrimax.

(Note: When typing in STDIN, the sequence CTRL+Z is considered end-of-file.)

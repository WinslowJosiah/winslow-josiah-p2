This page contains some examples of programs written in Gematrimax, as well as explanations as to how some of them work. In addition to functionality, many of these programs are written to be as interesting to read in English as possible.

Hello, World!

Lonely places. A hazardous friend. The handheld telephone keeps starting. This partially turned excessive. Tired, sleeping there again; beds beside clothes somewhere. This location's busy. Multiple minutes past, & still moving. After decades, will a connection resume?

This is perhaps one of the first programs most people write in any given language: a program that prints the phrase "Hello, World!". Luckily, it's a simple program to write in Gematrimax.

This works by pushing -1, pushing the codepoints of each character in reverse order, then looping through the items on the stack, outputting them until a value of -1 is reached.

Cat Program

We would demand information - a list including your characters - until this doesn't show another part.

This program copies its input directly to its output. It is equivalent to the following Cheat Mode program:

_[ "repeat until zero" , "ask for input" &#1+ "duplicate and add 1" &[ "if result not 0, this part runs" \. "output the original character" #1!] "end this 'loop'" ] "top of stack will be 0 iff done"

Basically, this consumes codepoints bring pushed onto the stack until adding 1 to them results in 0 - that is to say, the pushed "codepoint" is -1, marking the end of input.

Reverse Input

We write, & work backwards. A word stored somewhere absolutely changes structure! Words reversed themselves, & backwards text shortly became mixed.

Because data is pushed onto the stack with the first pushed item on the bottom, it is exceedingly easy to reverse input. All that needs to be done is, after each codepoint is pushed, output the top item of the stack until the stack is empty (checked here using the = command).

ASCII Loop

Human eyes indeed are monitoring a citizen. Business advertising sits over buyers, driving everything.

Fun fact: this program was the first program written in Gematrimax (though it has gone through many changes and iterations since then.) It uses a simple approach of incrementing a counter and outputting its value as a Unicode codepoint infinitely. One can easily loop through all 1,114,112 Unicode characters by modifying this program, but this is left as an exercise to the reader.

Fibonacci Numbers

Ideal greatness: a heavy term. A person's hurt itself with desperate feelings. This, babe, clearly reveals a problem.

This program demonstrates Gematrimax's capabilities of mathematical operations by outputting a the Fibonacci sequence. The sequence goes like this: the first term is 0, the second term is 1, and the next term is always the sum of the previous two terms.

Gematrimax's integers are of infinite precision; as such, this program will output Fibonacci numbers forever.

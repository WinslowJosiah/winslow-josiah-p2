What is Gematrimax?

Gematrimax is a programming language created in 2019 by Josiah Winslow, inspired by the Hebrew system of Gematria. Gematrimax works by converting regular English words into numbers, which each map onto one of 26 commands.

How do I write and run these programs?

There are two ways to run programs written in Gematrimax.

  1. Go to the Try It Online page, type your program into the given textbox, and click Run. (No installation needed!)
  2. Go to the Download page, download the given Python interpreter, and run a .gmx file with it. (This requires you to install Python.)

As far as writing Gematrimax programs, check out the docs section for a rundown of the commands and the way that programs are written. (Alternatively, you can try "running" a piece of regular text, and seeing what happens!)

Gematria? What's that?

Gematria is a system of turning words and phrases into numbers. While its origin is unclear among scholars, it was frequently used in Jewish culture, and it was likely influenced by the similar Greek system of isopsephy.

The Greek system of isopsephy involved assigning the first 9 letters of the alphabet values from 1 to 9, then the next 9 letters from 10 to 90, and then the next 9 letters from 100 to 900. Hebrew Gematria uses a similar system (although its alphabet is shorter, and thus its numbers end at 400). Many proponents of English-language Gematria use a system where the ordinal position of each letter (A=1, B=2 ... Z=26) is multiplied by 6 (for some reason) and added together.

Is this some kind of joke?

Umm...kinda?

Gematrimax is not meant to be a very serious programming language. However, it is a real programming language; the interpreter works, the example programs work, and it is indeed possible to write your own programs. Whether or not you would want to is another story.

How do I write and run these programs?

There are two ways to run programs written in Gematrimax.

  1. Go to the Try It Online page, type your program into the given textbox, and click Run. (No installation needed!)
  2. Go to the Download page, download the given Python interpreter, and run a .gmx file with it. (This requires you to install Python.)

As far as writing Gematrimax programs, check out the docs section for a rundown of the commands and the way that programs are written. (Alternatively, you can try "running" a piece of regular text, and seeing what happens!)

Gematria? What's that?

Gematria is a system of turning words and phrases into numbers. While its origin is unclear among scholars, it was frequently used in Jewish culture, and it was likely influenced by the similar Greek system of isopsephy.

The Greek system of isopsephy involved assigning the first 9 letters of the alphabet values from 1 to 9, then the next 9 letters from 10 to 90, and then the next 9 letters from 100 to 900. Hebrew Gematria uses a similar system (although its alphabet is shorter, and thus its numbers end at 400). Many proponents of English-language Gematria use a system where the ordinal position of each letter (A=1, B=2 ... Z=26) is multiplied by 6 (for some reason) and added together.

Gematrimax programs are too hard to write!

Of course, why wouldn't they be? It's not easy to make English words have certain letter sums for purposes of writing programs. However, I have made a utility called Gematria-Bot which lets you enter numbers and commands, and gives you suggestions of words that will fit for a Gematrimax program.

There is also an alternative syntax I created for the language, which I call Cheat Mode, that's much easier to understand and program in. You can use this syntax in the online interpreter by clicking the "Cheat Mode" button.

Why did you call it Gematrimax?

I called it Gematrimax mainly for two reasons:

  1. Programs written in Gematrimax are entirely controlled by their words' Gematria values. As such, the name represents something having a maximal quality of Gematria.
  2. In the system of English-language Gematria (where the ordinal positions of the letters are multiplied by 6 and added together), Gematrimax sums up to 666. (No, really; try it out!)
    • Coincidentally, Gematria-Bot has the same sum.
    • Cheat Mode has a sum of 444 under this system, which is kinda neat.

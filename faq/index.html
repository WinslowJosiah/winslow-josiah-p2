Is this some kind of joke?

Umm...kinda?

Gematrimax is not meant to be a very serious programming language. However, it is a real programming language; the interpreter works, the example programs work, and it is indeed possible to write your own programs. Whether or not you would want to is another story.

How do I write and run these programs?

There are two ways to run programs written in Gematrimax.

  1. Go to the Try It Online page, type your program into the given textbox, and click Run. (No installation needed!)
  2. Go to the Download page, download the given Python interpreter, and run a .gmx file with it. (This requires you to install Python.)

As far as writing Gematrimax programs, check out the docs section for a rundown of the commands and the way that programs are written. (Alternatively, you can try "running" a piece of regular text, and seeing what happens!)
